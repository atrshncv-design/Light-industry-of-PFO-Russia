# PFO Light Industry Map

This repo contains the PFO light industry interactive map as static HTML.

## Files
- `index.html` is the deployed entry (copied from `public/pfo-light-industry.html`).
- `public/pfo-light-industry.html` is the working full page.
- `public/pfo-light-industry-map.html` is the original map-only file from the archive; it contains a JS syntax error in `SVG_PATHS` and is kept for reference.

## Local preview
Open `index.html` directly in a browser.
